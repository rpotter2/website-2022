Animal Crossing Recipe Database - Flask & More

An application for users to sort and filter through recipes within the game Animal Crossing: New Horizons to find the most cost-effective item they can currently make or cost-ineffective recipes to avoid, as well as best gifts for villagers. Created for my Database class as a tour through multiple database structures including MariaDB, Redis, and MongoDB. The project also allowed me to flex my CSS muscles by mimicking the UI and aesthetics of the game.

Flask - IBM Poems

In Electronic Literature we learned one of the first American generated poems was Emmet Williams' "IBM Poem" from A Valentine for Noel which uses a process making poems that write themselves with a list of only 26 words. Unfortunately the original code is lost to time and improvements in tech, and doing it by hand is a tedious process. Therefore, I created this app using Flask to make it easier to experiment and play with.

Character Silhouette Generator - JavaScript

My final project for Applied Digital Studies was an application using pure JavaScript that randomly generates silhouettes intended for artists to practice designing diverse characters using basic shapes as is often used in animation and easily recognizable mascots. This was my first project that kicked off my interest in programming, and it means a lot to me.
